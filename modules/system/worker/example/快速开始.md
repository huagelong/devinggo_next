# Worker/Cron/Task 快速开始

## 1分钟上手新方式

### 创建一个Worker（处理任务）

```go
// modules/system/worker/server/my_worker.go
package server

import (
    "context"
    "devinggo/modules/system/pkg/worker"
    glob2 "devinggo/modules/system/pkg/worker/glob"
    "github.com/hibiken/asynq"
)

type MyData struct {
    Name string `json:"name"`
}

func init() {
    // 就这一行！
    worker.RegisterWorkerFunc("my:worker", executeMyWorker)
}

func executeMyWorker(ctx context.Context, t *asynq.Task) error {
    data, err := glob2.GetParamters[MyData](ctx, t)
    if err != nil {
        return err
    }
    glob2.WithWorkLog().Infof(ctx, "处理: %s", data.Name)
    return nil
}
```

### 创建一个Cron（定时任务）

```go
// modules/system/worker/cron/my_cron.go
package cron

import (
    "context"
    "devinggo/modules/system/pkg/worker"
    glob2 "devinggo/modules/system/pkg/worker/glob"
    "github.com/gogf/gf/v2/encoding/gjson"
    "github.com/gogf/gf/v2/frame/g"
)

type MyCronData struct {
    Param string `json:"param"`
}

func init() {
    // 就这一行！
    worker.RegisterCronFunc("my:cron", "我的定时任务", handleMyCron)
}

func handleMyCron(ctx context.Context, payload *glob2.Payload, params *gjson.Json) {
    if g.IsEmpty(params) {
        return
    }
    data := new(MyCronData)
    if err := params.Scan(data); err != nil {
        glob2.WithWorkLog().Errorf(ctx, "参数错误: %v", err)
        return
    }
    payload.Data = data
}
```

### 发送一个Task（触发任务）

```go
// modules/system/worker/task/my_task/my_task.go
package my_task

import (
    "context"
    "devinggo/modules/system/pkg/worker"
    "time"
)

type MyTaskData struct {
    Message string `json:"message"`
}

// 立即发送
func Send(ctx context.Context, message string) error {
    return worker.NewTaskBuilder(ctx, "my:worker").
        WithData(MyTaskData{Message: message}).
        Send()
}

// 延迟5秒发送
func SendWithDelay(ctx context.Context, message string) error {
    return worker.NewTaskBuilder(ctx, "my:worker").
        WithData(MyTaskData{Message: message}).
        WithDelay(5 * time.Second).
        Send()
}
```

## 就是这么简单！

- **Worker**: 1行注册 + 1个函数
- **Cron**: 1行注册 + 1个函数  
- **Task**: Builder链式调用

**代码量比传统方式减少90%！**

## 核心理念

### 关注业务逻辑
只需要关注：
- **Worker**: 执行逻辑是什么？
- **Cron**: 参数怎么解析？
- **Task**: 数据是什么？

### 无样板代码
不需要：
- ❌ 定义结构体
- ❌ 实现GetType()
- ❌ 实现GetPayload()
- ❌ 实现GetDescription()
- ❌ 实现GetCronTask()
- ❌ 创建构造函数

这些都自动帮你处理！
