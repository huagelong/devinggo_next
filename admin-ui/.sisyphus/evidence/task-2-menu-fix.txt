# Task 2 修复结果：effects/layouts 菜单文件编译错误

## 修复日期
2026-04-19

## 问题描述
根据 Task 1 诊断报告，3 个菜单文件存在 "Failed to resolve extends base type" 编译错误：
- `packages/effects/layouts/src/basic/menu/extra-menu.vue` (L12 `interface Props extends MenuProps`)
- `packages/effects/layouts/src/basic/menu/menu.vue` (L8 `interface Props extends MenuProps`)
- `packages/effects/layouts/src/basic/menu/mixed-menu.vue` (L13 `interface Props extends NormalMenuProps`)

## 根因
`packages/effects/layouts/package.json` 缺少 `@vben/tsconfig` devDependency，导致 pnpm 不创建 workspace 符号链接，TypeScript 无法正确解析 `extends: "@vben/tsconfig/web.json"`，进而导致 `@vben-core/menu-ui` 的类型定义无法解析。

## 修复步骤

### 1. 修改 package.json
在 `packages/effects/layouts/package.json` 中添加 devDependencies：
```json
"devDependencies": {
  "@vben/tsconfig": "workspace:*"
}
```

### 2. 安装依赖
运行 `pnpm install` 安装新添加的 devDependency。

### 3. 验证修复
运行综合验证脚本，确认：
- ✓ package.json 修改正确
- ✓ TypeScript 能正确解析 @vben/tsconfig
- ✓ MenuProps 类型定义解析成功
- ✓ NormalMenuProps 类型定义解析成功
